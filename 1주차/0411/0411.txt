  1. byte단위 입력
    - 숫자 입력(한자리 숫자)
    	int su = System.in.read()-48;
	         ---------
	          표준 입력 --> 키보드 입력

        키보드 입력        프로그램에서 받는 값 ---> (ASCII코드 값)
	    0      --->        48
	    1      --->        49
	    2      --->        50
	    3      --->        51
	    5      --->        53
	    9      --->        57


	 키보드 입력        프로그램에서 받는 값 ---> (ASCII코드 값)
	    a      --->        97
	    b      --->        98
	    c      --->        99
	    A      --->        65
	    B      --->        66
	    C      --->        67


	    
	    (자료형)데이터

	    System.out.println(su2);

	    double d2 = 300;	 //에러x
	    
	    System.out.ptintln(97);   ----->  97

  2. 라인 단위 입력 : 문자열과 10 이상의 수를 표현할 수 있음

       BufferedReader br = new BufferedReader(입력장치);
       String str = br.readLine();	str = "123"
       System.out.println(str);		123
       System.out.println(str + 100);	123100    문자열 + 숫자

   - 객체 생성
                                변환해서 전달 <-------- 얘한테 전달<--- 1번째 실행
       BufferedReader br = new BufferReader(new InputStreamReader(System.in));
       메소드호출리턴, 생성자, 추상클래스

   - 문자열 입력
     String str = in.readLine();	//a b c d 엔터 ===> "abcd"
     					//1 2 3 4 엔터 ===> "1234"
					//처음 입력한 문자부터 엔터 바로 이전에 입력한 문자까지를 문자열로 프로그램도 전달

     str+300 ?   1534(x) "1234300"(o)
